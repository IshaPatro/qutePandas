Setup

To begin using qutePandas, ensure you have a valid kdb+ license and the library installed. You can connect to your kdb+ session using the connect() method.

import qutePandas as qpd

qpd.connect()

By default, qutePandas looks for a license in ~/.qutepandas/ or a kdb_lic/ folder in your project root.
