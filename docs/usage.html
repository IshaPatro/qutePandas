How to use these functions

Data Cleaning

Functions to handle null values and duplicates with high-speed kdb+ execution.

cleaned = qpd.dropna(q_table)

cleaned_col = qpd.dropna_col(q_table, col="col_0")

filled = qpd.fillna(q_table, col="col_0", value=0)

unique = qpd.remove_duplicates(q_table)

Transformation

Structural changes and type casting.

renamed = qpd.rename(q_table, {"old": "new"})

casted = qpd.cast(q_table, col="col_0", dtype="float32")

subset = qpd.drop_col(q_table, cols=["col_9"])

Grouping & Aggregation

High-performance aggregations leveraging q engines.

summary = qpd.groupby_sum(q_table, by_cols="cat", sum_col="val")

avg_stats = qpd.groupby_avg(q_table, by_cols="cat", avg_col="val")
